JoJo Personality Quiz é uma aplicação web desenvolvida com HTML, CSS e JavaScript que identifica o personagem de *JoJo's Bizarre Adventure* mais compatível com a personalidade do usuário.
O projeto utiliza um sistema próprio de perguntas e respostas baseado em atributos comportamentais, como bondade, inteligência, coragem, determinação, empatia, ambição, manipulação, honra, 
entre outros. Cada resposta altera esses atributos e, ao final do questionário, um algoritmo compara o perfil do usuário com uma base de dados contendo dezenas de personagens das oito partes da obra.
Como resultado, o sistema apresenta o personagem mais compatível, além de um ranking com os três personagens de maior semelhança, acompanhado de informações como parte, Stand, descrição e imagem.

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento web, incluindo manipulação do DOM, eventos, estruturas de dados, algoritmos de comparação, organização 
de código em múltiplos arquivos e criação de interfaces responsivas.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## Funcionalidades

- Sistema de perguntas baseado em atributos.
- Mais de 50 perguntas de personalidade.
- Comparação com dezenas de personagens de todas as partes de JoJo's Bizarre Adventure.
- Exibição do personagem mais compatível e ranking dos três melhores resultados.
- Interface responsiva e intuitiva.
- Hospedagem via GitHub Pages.
